## Watch 모드

터미널 두개를 실행시킨다.

```shell
cargo watch
```

```shell
cargo watch 'make serve'
```

이렇게 하면 build가 되는 족족 서버에 곧바로 적용된다. 하지만 터미널을 두 개 띄워야하는데... nodemon 사용하는 것처럼 해봤는데... 이게 맞냐?
